Street Fighter-Style Game (Python)

As the team lead for my final project in 15-122: Fundamentals of Programming, I spearheaded the development of a 2D fighting game inspired by Street Fighter, implemented entirely in Python using the cmu_graphics library.

I coordinated the design and division of labor within the team, managed our development timeline, and personally implemented core game features such as character movement, collision detection, hitbox logic, and animation switching. The project also involved sound integration, sprite management, and basic AI behavior for a non-player character.

This project strengthened my skills in object-oriented programming, real-time input handling, and collaborative software development. It was an exciting challenge to bring a dynamic game environment to life from scratch using foundational Python skills.
